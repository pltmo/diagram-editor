Signal Description (JSON)
Wave chars:
0 low  1 high  h/l same
z hi-Z  x unknown  . continue
p/P rising-edge clock  n/N falling-edge clock
capitals add an edge marker arrow  . repeats
= bus+label  2–9 colored bus+label  d continue
labels consumed in order from "data":[]

Timing arrows:
"edge": ["A→B label"]
"node": {"A":{"signal":"CLK","cycle":2}}

Per-signal: "period" cell width multiplier  "phase" x offset
Timing:
"timing": {"unit":"ns","period":10,"trise":1}
units: fs ps ns us ms s
"trise" sets bus/X transition width in same units — scales chamfer with zoom, makes measure tool snap to stable-data edges
